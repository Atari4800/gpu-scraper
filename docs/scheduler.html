scheduler module¶

This module primarily contains the Scheduler class, responsible for creating cronjobs. If it is run as a script, then it will create a cronjob to run initiator.py regularly. The script would need an integer command line argument to indicate how frequently initiator.py should be run.

class scheduler.Scheduler(minutes)¶

Bases: object
