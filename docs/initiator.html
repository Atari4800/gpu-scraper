initiator module¶

This file handles the ‘main’ for the backend of the gpu-scraper application. It is able to ping hosts, and controls how a productList is searched through for content.

initiate()¶

Begins the backend portion of the program, loads the JSON file and reads for the urls therein. If the url contains one of the supported link websites, then it calls the appropriate subprocess to attempt to find product in the url. If a product is found/not found isAvailable is updated accordingly. If a product is found then lastAvailable is updated with the current date/time.

Returns

The number of websites that were crawled.
