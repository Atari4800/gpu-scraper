This file handles the adding, and deleting of items from the productList.JSON file.

class item_base.item_base¶

Bases: object

This class handles the modification of items inside of a designated productList.

static add_item(url, title, price, json_file)¶

Adds an item to the designated product list as a JSON entry. First it checks a URL for product information, then it adds corresponding price, and URL information to the productList.

Parameters

  • url (string) – The url of the item that needs to be monitored (B&H, Newegg, and Bestbuy links only)

  • title (string) – The name of a product to be added.

  • price (double) – The price of a product to be added.

  • json_file (string) – The productList that the URL’s JSON entry will be placed in.

Returns

-6 if the price cannot be found.

Returns

-5 if there is a duplicate link found within the JSON file.

Returns

-4 if the URL entered is not supported.

Returns

-3 if the URL’s domain cannot be reached.

Returns

-2 if there is a problem opening the JSON file or Reading from it.

Returns

-1 if the item cannot be found.

Returns

0 if the item could not be added to the file.

Returns

1 if the item was added successfully.

static del_item(url, json_file)¶

Deletes a specified JSON entry from a specified json_file by URL

Parameters

  • url (string) – The url of the item to be deleted.

  • json_file (string) – The file to delete the JSON entry from.

Returns

0 if an error occurred when trying to delete the entry.

Returns

1 if the item was deleted successfully.
