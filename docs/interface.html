interface module¶

This file handles the GUI component of our application. It is responsible for creating the main GUI and related components like popup windows or notifications.

class interface.ProductGrid(master)¶

Bases: tkinter.Frame

One instance of this class handles all data and visual components for the query table in the main interface screen. It creates a grid of information, where each row has the information and buttons for a single product.

Creates a grid in the main interface page.

Parameters

master (string) – The tkinter container which is a master of this grid.

add_row()¶

Adds a row to the table. The object has data to keep track of which product should be displayed in the next row.

Returns

returns nothing

delete_row(row)¶

Deletes the given row from the table.

Parameters

row (A List) – An entire row in the table. It should be a List with all of the components in that row.

Returns

returns nothing

load_json()¶

Updates this instance’s data from the file productList.json

Returns

returns nothing

refresh()¶

This is a function that is responsible for updating the content inside the Grid of information in the GUI. It has an initial filler function, but should be assigned a meaningful function later on after the meaningful one is written. (It should be assigned to update_scroll_region after it is defined)

interface.launch_gui()¶

Launches the main GUI window for the application. It displays information about current queries and allows the user to work with the queries (adding, deleting, searching for them instantly), and has a button to create the cronjob which would automatically search for product after regular intervals of time.

Returns

returns nothing

interface.notification(product, source, price, belowMSRP)¶

Creates a notification window in the bottom right hand side of the user’s screen. It will display the given information and does not close automatically.

Parameters

  • product (string) – The product available

  • source (string) – The url of the product

  • price (string) – The price of the product

  • belowMSRP (string) – Whether or not the product is below its MSRP

Returns

returns nothing

interface.shorten_url(url)¶

This standalone function will attempt to return a shorter version of the given url. This is useful for displaying the source of a query to the user for a situation like a GUI or notification where the full url might require too much space.

Parameters

url (string) – The url to attempt to shorten

Returns

If the url is for one of the websites we are focusing on, it returns the name of that business.

A best buy url would return ‘Best Buy’ A Newegg url would return ‘Newegg’ A B&H url would return ‘B&H’

Otherwise, would return ‘Other’
