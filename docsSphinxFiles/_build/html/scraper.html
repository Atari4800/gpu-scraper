scraper module¶

This script runs a web scraper and searches for specific parameters within a webpage.

class scraper.Scraper(url, url_type)¶

Bases: object

This class handles web scraping operations.

Instantiates a web scraper and pulls initial page source information.

Parameters

  • url (string) – The url of a link that will be scraped to find product information

  • url_type (string) – The type of link that a url is BB - Bestbuy NE - Newegg BH - B&H

get_bb()¶

Searches for whether or not the HTML source grabbed by the scraper contains a product available for purchase with BestBuy

Returns

0 if the product is not found available for purchase.

Returns

1 if the product is found and available for purchase.
